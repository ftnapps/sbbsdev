Synchronet BBS on Debian systems

Synchronet is a BBS system that supports multiple simultaneous users with hierarchical message and file areas, multi-user chat, and BBS door games. This project enables the development of it on Debian or Debian derived systems like Ubuntu, for the creation of Synchronet BBS Debian packages and/or just compileing the source for local installation.

    Branches
  • The master branch in the repository contains the current/active versions of the scripts and directories.
  • The develop branch in the repository is used for developing new versions of the scripts and directories.
